# Global Timezone Social Clock

This project is a fully functional social clock application that displays the current time in various timezones around the world. It provides users with a platform to share thoughts and updates with others, thus creating social interaction based on time.

## Features
- Display current time in selected timezones
- User-generated posts
- Responsive design
- Integration with Supabase for user authentication and database management

## Tech Stack
- Next.js (TypeScript)
- Tailwind CSS
- Supabase
- React

## Installation Instructions
1. Clone the repository
2. Run `npm install` to install dependencies
3. Set up your Supabase project and database. 
4. Create a `.env` file using the `.env.example` file and insert your Supabase keys.
5. Run `npm run dev` to start the development server.

## Supabase Setup
- Create a new Supabase project
- Use the provided schema in the Supabase UI to set up your database

## Deployment Instructions
Deploy your application using Vercel, Netlify, or your hosting choice.
